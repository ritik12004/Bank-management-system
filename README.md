# Bank Management System (Python OOP)

A simple **Bank Management System** built using **Python** to demonstrate the concepts of **Object-Oriented Programming (OOP)**.  
This project allows users to create a bank account, deposit or withdraw money, and check their balance — all through a simple command-line interface.

---

## Features
- Create a bank account with holder name and account number  
- Deposit money into the account  
- Withdraw money (with balance validation)  
- Check current account balance  
- Simple and beginner-friendly code structure  

---

## Concepts Used
- Classes and Objects  
- Constructor (`__init__` method)  
- Encapsulation (data protection using methods)  
- Conditional statements and loops  

---

## Technologies Used
- **Language:** Python 3  
- **Libraries:** No external libraries required
